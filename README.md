# The Prismatic Veil

A wave-survival action game built with Phaser 3 — no build tools, no frameworks, just a single self-contained HTML file.

**Play now:** https://jr-faulkner.github.io/Prismatic-Veil/

## Characters

| Character | Title | Combat Style |
|---|---|---|
| Prismel | Prism Weaver | Ranged prism shots with spread |
| Kineza | Momentum Born | Fast melee striker |
| Auryi | Aura Spoken | Area aura caster |

## Resonance Shards

On level-up, choose a Resonance Shard to empower your run:

| Shard | Frequency | Effect |
|---|---|---|
| Shard of Ardence | Red | Strike force intensifies |
| Shard of Current | Green | Momentum flow accelerates |
| Shard of Resonance | Yellow | Attunement sharpens |
| Shard of Reach | Blue | Spectral projection extends |
| Shard of Endurance | Orange | Strand-link reinforced |
| Shard of Mending | Violet | Strand-fabric rewoven |

## Controls

**Desktop:** WASD or Arrow Keys to move
**Mobile:** Touch and drag to move

## Built With

- [Phaser 3](https://phaser.io/) (embedded inline, no CDN)
- Web Audio API for synthesized sound effects
- Plain HTML/JS — no build step, no dependencies
